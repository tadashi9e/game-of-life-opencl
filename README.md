# game-of-life-opencl
A OpenCL implementation of Conway's Game of Life

## Requirement
* OpenCL >= 1.1
* OpenGL >= 4.4
* X11
* gcc >= 4.9

## Platform
* Linux

## Building
```
make
```

## Execution

```
./gol [-w WIDTH] [-h HEIGHT] [-f file]
```

 -w : field width
 -h : field height
 -f : Life1.05 format file
